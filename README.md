# RaceIQ v0.1.0

First build of **RaceIQ** — a sim-racing telemetry analysis and coaching app for Windows. Drive, and RaceIQ records every lap, breaks it down corner by corner, tells you where you're losing time, and ranks your best laps on a live leaderboard.

## What's inside
- **Live telemetry** — real-time speed, throttle/brake, gear and inputs as you drive.
- **Lap analysis** — distance traces, a track map with micro-sectors, and sector times for every recorded lap.
- **AI coach** — corner-by-corner feedback on where the time goes, fully offline.
- **Online leaderboard** — valid laps upload automatically and rank globally or among friends, per car + track.
- **Friends** — share your code, add friends, and import their laps to compare against your own.
- **Auto-updating** — the app updates itself, so you're always on the latest build.

Local analysis (track map, traces, coach) works with no account and no internet — only the leaderboard goes online.

## Install
1. Download **RaceIQ-Setup-0.1.0.exe** below and run it (installs per-user, no admin needed).
2. Windows SmartScreen will warn because the app isn't code-signed yet — click **More info → Run anyway**. It's safe; signing is on the roadmap.
3. Launch **RaceIQ** from the Start menu.

## Get on the leaderboard
1. Open the **RANKS** tab and create an account (email + password).
2. Hop into **Assetto Corsa Competizione** (Le Mans Ultimate also supported) and drive a clean lap — it uploads automatically and shows up on the board.
3. Add a friend by their code to compare laps head-to-head.

## Requirements
- Windows 10/11 (64-bit)
- For live capture: Assetto Corsa Competizione or Le Mans Ultimate
